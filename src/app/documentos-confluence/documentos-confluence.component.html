Documentos de Confluence

Como herramienta colaborativa cuenta con una gran cantidad de tipos de documentos para diferentes tipos de propositos en una organizacion, algunas de ellas pueden generar algun tipo de costo pero cuentan con una gran variedad de herramientas libres

TIPO DE DOCUMENTOS

Pagina en Blanco Este tipo de documento es el mas libre que existe y permite agregar todo tipo de contenido soportado por el lenguaje html Entrada de Blog Se asemeja al funcionamiento de un blog, documento facil de compartir de forma externa Analisis DAFO Documento que permite evaluar datos por medio de formulas como las aplicadas en excel Analisis Ejecutivo Documento que se puede alimentar de integraciones para generar informes de rendimiento de alto nivel

COMPONENTE DE UTILIDAD

Confluence cuenta con una poderosa utilidad tipo servicio que es reutilizable, un ejemplo de sus amplia gama de funcionalidades seria Para crear los documentos enlazados con Jira debemos iniciar con el carácter de / y esto nos permite desplegar un motor de componente donde podemos instalar más apartados, tecleamos Bitbucket y automáticamente nos salen las incidencias que están en Jira para poder enlazarlas, confluence también nos permite comentar el código gracias a sus componentes
