Junit

¿ Que es ?

Las pruebas unitarias del lado del servidor aprueban si el comportamiento lógico de un objeto(clase) es el correcto, a través de la implementación de test La detección de errores en fases tempranas es gracias a la implementación de estas pruebas, permitiendo ahorrar dinero y tiempo Junit es un framework de automatización de pruebas unitarias que funciona en el lado del servidor Junit fue diseñado específicamente para ejecutarse con Java, el objetivo de Junit es evaluar cada componente de una clase clase para evitar generar errores Cuando se pretende comprobar el funcionamiento efectivo de un bloque de código, debemos ejecutar pruebas unitarias, este proceso es mayormente utilizado en metodologías agiles
