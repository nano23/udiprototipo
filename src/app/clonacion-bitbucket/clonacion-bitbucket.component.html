Clonación del Repositorio

La clonación es un proceso que permite crear una copia local identica de las fuentes del repositorio en la nube, con la finalidad de realizar modificaciones que posteriormente seran integradas con el codigo original
En este apartado se vera un ejemplo de como se realiza una clonación de forma exitosa

Ejemplo clonación

En este ejemplo practico se describe un pequeño paso a paso de como realizar una clonación de repositorio principal a local

Requisito Previo

Es necesario descargar e instalar el software Git en el equipo local en el que se realizara la clonación

Es posible descargarlo desde la url Git descargar

1) Copiar el comando con la URl del repositorio a clonar desde la herramienta de bitbucket

Conexión

2) Se debe implementar el comando de git que nos permite clonar el repositorio en ambiente local, es necesario establecer un canal de comunicación entre nuestro ambiente local(Git) con el ambiente remoto(Bitbucket), estos canales de comunicación pueden ser por llaves públicas y privadas conocidos como SSH o pueden ser por protocolo HTTP, en esta imagen veremos cómo Bitbucket nos despliega una interfaz para el usuario y contraseña con la finalidad de comprobar la conexión entre local y remoto
