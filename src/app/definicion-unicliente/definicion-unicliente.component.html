Karma/Jasmine

¿ Que es ?

Las pruebas unitarias del lado del cliente determinan la funcionalidad del código, a través de test que verifican si el comportamiento y la lógica son correctos Jasmine es una herramienta que nos permite realizar pruebas unitarias, se puede ejecutar en diversos tipos de plataforma, Jasmine sobre sale gracias a que permite realizar pruebas unitarias de forma asíncrona El framework de angular implementa por default la herramienta de pruebas unitarias Jasmine, gracias a esto podemos realizar pruebas de forma nativa y las dos tecnologías van a correr de forma armónica Es importante resaltar que la sintaxis de Jasmine es muy simple y minimalista, además cada test creado se cuenta como una función propia del usuario. Jasmine también cuenta con suites, specs, expectations, matchers Jasmine permite crear las pruebas, funciona como un framework de desarrollo de pruebas en el lenguaje typescript Karma se encarga de leer y ejecutar las pruebas creadas en Jasmin, generar reportes con los resultados y ejecutar procesos en fase de ejecución
