Ejemplo Grabación usando blazemeter

Blazemeter permite configurar el contenido que queremos capturar en la prueba de carga, ejemplo, solo capturar archivos con extension html, imagenes o javascript Blazemeter permite exportar las pruebas en formatos legibles para sofware como Jmeter o Selenium Blazemeter minimiza el esfuerzo necesario en la construcción de pruebas de carga
