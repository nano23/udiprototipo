sonarqube

¿ Que es ?

Plataforma web que funciona para la gestión del código, su aplicación es principalmente para código Java, pero se puede implementar en cualquier otro tipo de lenguaje La función de SonarQube se basa en la generación de reportes basados en más de 600 reglas que analizan el código en temas como Violaciones de código, análisis dinámico de capa por capa e historial de datos. Funciona como un centro de calidad en el ambiente de desarrollo Permite tener una inspección continua del código con el fin de lograr escalar las necesidades de los proyectos Los análisis sobre el estado del sistema se actualizan de forma relacional a las nuevas versiones de código Ayuda a medir la calidad con la que el código está siendo escrito aplicado a temas como código duplicado, código muerto, estándares de codificación, bugsError de software, comentarios, complejidad ciclomática, pruebas unitarias y cobertura de código
