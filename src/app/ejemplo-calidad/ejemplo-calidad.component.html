Para obtener un resumen un poco más detallado, se pulsa sobre la palabra que Prototipo-Udi, la herramienta permite que se pueda navegar a través de cada uno de sus módulos bugsError de software, Vulnerabilities, Security entre otras, también es importante notar que Sonarqube se integra con Jacoco Los bugsError de software son errores en el software, que generan conflictos al momento de ejecutar el código, también se aplica este término como un desencadenador de resultados inesperados Seleccionar en el menú, el apartado de Code Smell, va a permitir gestionar un reporte donde se podrá observar como la herramienta de Sonarqube interpreta el código que se puede mejorar, en cuanto a las buenas prácticas del código Se integra con jacoco para presentar el reporte de cobertura de codigo Sonarqube permite visualizar la cobertura del código, la diferencia de esta cobertura con la que brinda Jacoco, es que estos reportes se pueden compartir con todo el grupo de desarrollo o personas implicadas en el proyecto El código duplicado también es objeto de identificación por medio de esta herramienta, gracias a la detección del código duplicado se pueden reutilizar métodos por medio del cambio de parámetros. Cada vez que solucionamos algo en el entorno de desarrollo del proyecto, se podrá hacer huso de una nueva ejecución sobre el proyecto, cada nueva ejecución brinda un nuevo reporte donde podremos compararlos con reportes anteriores.
