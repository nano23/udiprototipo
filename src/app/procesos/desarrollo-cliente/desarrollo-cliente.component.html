Desarrollo del lado del Cliente

Front-End

Bienvenido, en este apartado se dara a conocer los aspectos relevantes para desarrollar el front-end del prototipo

package.json : Archivo de definición de nodejsentorno en tiempo de ejecución multiplataforma, de código abierto en el lenguaje javascript, en este archivo se detallan el nombre del proyecto, version, scriptssecuencia de comandos en un lenguaje legible por una maquina de ejecución y dependencias del proyecto

angular.json : Archivo principal de configuración de un proyecto angular, en el se definen propiedades para las diferentes etapas de construcción del proyecto

index.html : Archivo de ejecución del proyecto

environment.ts : Permite especificar variables que controlaran funciones segun el ambiente de ejecucion, por lo general pruebas/producción

jwt.interceptor.ts : tipo de elemento de angular que permite crear intermediarios de procesos, ejemplo, mecanismo que intercepte todas las peticiones http que salgan de la aplicación

pagina1.component.html : pagina ejemplo creada para la construcción del prototipo, se construye apartir de html, css y elementos de angular

pagina1.component.ts : archivo angular en lenguajetypescript que controla el comportamiento de la pagina pagina1.component.ts

pagina1.component.css : documento con los estilos a aplicar en el componente de pagina web pagina1.component.cs
