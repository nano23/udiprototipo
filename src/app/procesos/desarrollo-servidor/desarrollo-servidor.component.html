Desarrollo del lado del Servidor

Back-End

Bienvenido, en este apartado se dara a conocer los aspectos relevantes para desarrollar el back-end del prototipo

pom.xml : Archivo de definición de dependencias de maven, este archivo principal permite identificar todas las tecnologias, configuraciones y generales del proyecto java

PrototipoApplication.java : Clase que cuenta con el metodo static void main que permite levantar la aplicación como servicio

Pagina1RestController.java : Documento que funge como controlador tipo rest de los servicios http expuestos para el control de la clase pagina1

Pagina1Dto.java : Clase del objeto pagina1 de transferencia de datos visible/usable en la pagina proveniente desde la base de datos

Pagina1.java : Clase que funge como entidad de base de datos de parte del servidor, no debe ser visible por la aplicación cliente
