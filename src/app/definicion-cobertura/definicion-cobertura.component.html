Jacoco

¿ Que es ?

Cobertura de código es un indicador de medida que notifica cuántas líneas / bloques / arcos de su código se ejecutan mientras se lanzan las pruebas automatizadas. La cobertura del código se recopila mediante el uso de una herramienta especializada para instrumentar los binarios a fin de agregar llamadas de seguimiento y ejecutar un conjunto completo de pruebas automatizadas contra el producto instrumentado Jacoco realiza los seguimientos y genera un reporte HTML dentro de un entorno local Dicho reporte lo podemos visualizar en nuestro navegador, esta herramienta nos permite ver cada archivo del proyecto, permitiendo verificar paso a paso la cobertura de cada paquete Si en el reporte de Jacoco entramos a un paquete especifico podremos ver las líneas de código recorridas, cuales fueron los escenarios analizados y cuáles son las rutas que aún no tienen cobertura Se usa por medio de anotaciones java en el codigo de pruebas unitarias
