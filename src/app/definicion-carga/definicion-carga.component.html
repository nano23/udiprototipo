Definición

Bienvenido, en este apartado se dara un acercamento sobre la herramienta seleccionada para las pruebas de carga

Basado en un proceso de benchmarkingbenchmarking es un proceso de crear, recopilar, comparar y analizar indicadores claves que permitan medir el rendimiento de los procesos las herramientas seleccionadas segun la arquitectura del prototipo propuesta serian

Jmeter/Blazemeter/Taurus

¿ Que es ?

Tienen como objetivo general, observar el comportamiento de una aplicación o proyecto Se observa el rendimiento bajo una cantidad de peticiones determinada, los usuarios podrían ser concurrentes y se puede especificar qué tipos de tareas serán las ejecutadas Uno de los factor mas importante es determinar el tiempo de ejecución de la prueba Para el prototipo web se utilizarán tres herramientas para poder cumplir estas pruebas de carga BlazeMeter,Jmeter y taurus BlazeMeter es un todo un marco de trabajo pero para nuestro caso utilizaremos el plugin del navegador chrome para capturar las acciones realizadas sobre la pagina web del prototipo La otra herramienta de pruebas de carga es Jmeter,esta, se descarga y se instala como cualquier programa, el link para descargarlo es https://jmeter.apache.org/ Con Jmeter se pueden construir manualmente pruebas de carga asi como tambien importar pruebas generadas desde herramientas como BlazeMeter para hacerles modificaciones y ejecutarlas La ultima herramienta de pruebas de carga es Taurus,esta, se descarga y se instala facilmente, el link para descargarlo es https://gettaurus.org/ Finalmente con la herramienta Taurus es posible ejecutar scriptssecuencia de comandos en un lenguaje legible por una maquinasecuencia de comandos en un lenguaje legible por una maquina de pruebas en formayo yml creados con anterioridad por herramientas como blazemeter o jmeter Los scripts de Taurus se usan para complementar procesos DevOpsGrupo de prácticas que unifica operaciones de TI y desarrollo de software, como por ejemplo permitir ejecucion en segundo plano de pruebas de carga sobre servidores en la nube de blazemeter
