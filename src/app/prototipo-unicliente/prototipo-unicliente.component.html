Prototipo

Para el desarrollo del prototipo se escogieron las herramientas(frameworks) de Jasmin y karma, debido a que angular es el marco de trabajo con el que se implementó el lado del cliente Las pruebas unitarias del lado del cliente hacen parte del engranaje DevOpsGrupo de prácticas que unifica operaciones de TI y desarrollo de software Por este motivo y para este ejemplo en particular es necesario que se realicen de forma satisfactoria para poder finalizar los procesos de pipeline en el proceso de orquestación Las pruebas construidas son basicas con el objetivo de dar a conocer el funcionamiento de la herramienta y su integración, a continuación se presenta un ejemplo de codigo desarrollado
