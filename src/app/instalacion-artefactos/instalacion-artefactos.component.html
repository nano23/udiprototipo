Instalación y configuracion del reposotorio de artefactos

Bienvenido, en este apartado se dara un acercamento a los procesos de instalación y configuración en la herramienta Nexus Repository

Descarga

Para descargar el contenido de instalación se debe dirigir a la pagina Nexus Repository

En cualquier sistema operativo que se pretenda instalar y configurar, nexus funciona como un sistema independiente y par replicarlo en otra maquina solo basta con copiar y pegar todo su contenido

Pagina Descarga

Ejecución

Una vez descargado el Nexus Repository, se tendrá acceso al archivo ejecutable para su correspondiente instalación

Para poder inicializar la herramienta, se copia la ruta y se ejecuta el comando de instalación nexus.exe /run

File System

Ejecución

Interfaz

Finalizada la ejecución del Nexus Repository, se inicializa una interface web en donde podemos gestionar todos nuestros repositorios de artefactos Para poder inicializar la herramienta, se debe proceder a realizar su respectivo Login, el primer ingreso tiene por default el nombre de usuario admin y la contraseña se encuentra en la ruta indicada por la interface

Interface

Login

Configurar Nuevo Repositorio

Finalizado el Login, se vera la siguiente interfaz, debes crear un nuevo repositorio, esto se logra por medio del icono del engranaje de color verde Ahora se debe seleccionar algún tipo de repositorio para ser creado, para el proyecto se escoge el repositorio de Maven2(hosted), esta selección es debido a que lo que se realizara es guardar en el repositorio por ende seleccionados hosted Después de selección el Maven2 (hosted), se despliega una nueva interfaz en donde se debe seleccionar alguno de los repositorios que por defecto crear la herramienta, para el proyecto se selecciona Maven-snapshots
